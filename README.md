# invitation-to-date-


Этот сайт был сделан для приглашения девушки на свидание, чтобы увидеть что она выбрала вам всего лишь надо вставить свой chat id и 
bot token от личного телеграм бота.

🤖 Как создать бота в Telegram (пошагово)

Шаг 1. Найдите @BotFather

· Откройте Telegram.
· В поле поиска введите @BotFather (это официальный бот для создания ботов).
· Нажмите на него и начните диалог (кнопка «Start» или команда /start).

Шаг 2. Создайте нового бота

· Напишите команду:
  ```
  /newbot
  ```
· BotFather попросит придумать имя бота (отображается в списке контактов, например, «Моё свидание»). Введите его.
· Затем попросит придумать username (уникальный идентификатор, должен заканчиваться на bot, например MyDatingBot или YourNameInviteBot). Если имя занято, BotFather предложит другое.
· После успешного создания вы получите сообщение с токеном – это длинная строка вида:
  ```
  1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
  ```
  Сохраните этот токен – он нужен для подключения бота к сайту.

Шаг 3. Получите свой Chat ID

Chat ID – это уникальный номер вашего аккаунта (или группы/канала), куда бот будет присылать уведомления.

· Напишите своему боту любое сообщение (например, «Привет»). Это важно, чтобы бот «увидел» вас.
· Затем откройте в браузере ссылку (подставьте свой токен вместо ВАШ_ТОКЕН):
  ```
  https://api.telegram.org/botВАШ_ТОКЕН/getUpdates
  ```
  Например, с вашим токеном:
  ```
  https://api.telegram.org/bot8710199001:AAEFt4NjlKomlXcAINw80LThkOZSJUPfFuI/getUpdates
  ```
· Вы увидите JSON-ответ. Найдите внутри поле "chat" и под ним "id". Это число и есть ваш Chat ID.
  Например:
  ```json
  {"ok":true,"result":[{"update_id":123456789,
    "message":{"message_id":1,"from":{"id":123456789,...},
    "chat":{"id":1454092601,"first_name":"Ваше имя",...},
    ...
  }]}
  ```
  В этом примере Chat ID = 1454092601.

Если вы не видите "chat" – возможно, вы ещё не написали боту. Напишите ему любое сообщение и обновите страницу.

Шаг 4. Вставьте токен и Chat ID в код вашего сайта

В файле index.html найдите строки:

```javascript
const BOT_TOKEN = 'ВАШ_ТОКЕН_ОТ_BOTFATHER';
const CHAT_ID = 'ВАШ_CHAT_ID';
```

Замените их на свои данные.

✅ Проверка работоспособности

Чтобы убедиться, что бот работает и сможет принимать сообщения:

· Откройте диалог с вашим ботом в Telegram.
· Отправьте ему любое сообщение (он пока не ответит, но это активирует чат).
· Теперь, когда на сайте кто-то нажмёт «Сохранить место», бот пришлёт вам сообщение с данными.
